# Anime Popularity Analysis 

## Project Objective
To perform exploratory data analysis on Anime dataset and analyze factors affecting popularity.

## Tools Used
- Python
- Pandas
- Matplotlib
- Seaborn

## Analysis Performed
- Data Cleaning
- Handling Missing Values
- Distribution Analysis
- Popularity vs Duration Visualization
- Correlation Analysis

## Key Insights
- Most anime belong to TV format
- Popularity shows slight relation with duration
- Few extreme outliers present in dataset

##  Dataset Source
Kaggle Anime Dataset
